# College Event Management System

## College Event Management (MERN + AI Recommendations)

A complete **MERN stack College Event Management web application** with role-based access and AI-powered event recommendations.

### Features

- **Student role**
  - Browse upcoming events
  - Register and track registered events
  - Get event recommendations based on interests

- **Society / Club Admin role**
  - Create, update, and manage events
  - View student registrations

- **AI-powered recommendations**
